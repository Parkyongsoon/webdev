웹 개발 실습
=========

  ##### 공부한 내용을 토대로 하나씩 만들어보는 공간.

사용기술
======
  
  * Node.JS 14.4.0
  * Koa 2.13.0
  * Postgresql 13.1
  * Redis 3.0.2

실행
===

  * 설치   
    yarn
  
  * 실행   
    yarn start:dev
